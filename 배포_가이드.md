# 웹 링크로 배포하기 (Streamlit Community Cloud)

이 앱을 무료로 인터넷에 올려 **URL 하나로 공유**하는 방법입니다.
상대방은 설치 없이 브라우저로 그 링크만 열면 됩니다.

> 준비물: **GitHub 계정** (무료, https://github.com 에서 가입)

---

## 1단계 — 코드를 GitHub에 올리기

### 방법 A. GitHub Desktop (권장, 명령어 불필요)

1. https://desktop.github.com 에서 **GitHub Desktop** 설치 후 로그인.
2. 상단 메뉴 **File → Add local repository** → 이 폴더(`임피던스 피팅`) 선택.
   - 이미 git 저장소라 그대로 인식됩니다.
3. 오른쪽 **Publish repository** 클릭.
   - 이름 확인 후, 비공개로 하려면 **"Keep this code private"** 체크 (Streamlit Cloud는 비공개도 배포 가능).
   - **Publish** 클릭 → GitHub에 업로드 완료.

### 방법 B. 브라우저 업로드 (도구 설치 없이)

1. https://github.com/new 에서 새 저장소 생성 (이름 예: `impedance-fitting`).
2. 생성된 페이지의 **uploading an existing file** 링크 클릭.
3. 이 폴더의 파일들을 드래그해서 업로드 (`.git` 폴더는 제외, 나머지 전부).
4. **Commit changes** 클릭.

---

## 2단계 — Streamlit Cloud에서 배포

1. https://share.streamlit.io 접속 → **Continue with GitHub** 로 로그인(권한 허용).
2. **Create app → Deploy a public app from GitHub** (또는 "Deploy an app").
3. 세 가지만 지정:
   - **Repository**: 방금 올린 저장소 선택
   - **Branch**: `master`
   - **Main file path**: `app.py`
4. **Deploy** 클릭 → 몇 분간 설치가 진행되고, 끝나면 `https://...streamlit.app` 형태의 **URL**이 나옵니다.

---

## 3단계 — 공유

- 나온 URL을 그대로 전달하면 됩니다. 상대는 브라우저로 열기만 하면 사용 가능.

---

## 참고사항

- **자동 재배포**: 앞으로 코드를 GitHub에 올릴 때마다(=push) 앱이 자동으로 업데이트됩니다.
- **슬립**: 무료 앱은 한동안 아무도 안 쓰면 잠들고, 다시 접속하면 약 30초 뒤 깨어납니다.
- **비공개 저장소**도 배포할 수 있지만, 앱 URL을 아는 사람은 접속할 수 있습니다(코드는 비공개, 앱은 공개). 접근 제한이 필요하면 Streamlit Cloud 설정에서 **뷰어 이메일 제한**을 걸 수 있습니다.
- **데이터 프라이버시**: 사용자가 업로드한 `.z` 데이터는 처리 과정에서 Streamlit 서버로 전송됩니다. 민감한 연구 데이터라면 이 점을 유념하세요(로컬 실행이 필요하면 폴더 복사 방식 사용).
- **Python 버전/패키지**: `requirements.txt` 를 Cloud가 자동으로 읽어 설치합니다. 문제가 생기면 앱 설정의 **Advanced settings → Python version** 을 3.12 로 지정해 보세요.
